# Load Balancer Project

This folder contains scripts for the Load Balancer project in ALU System Engineering & DevOps.

## Files

- `0-custom_http_response_header` : Bash script to configure Nginx with a custom HTTP response header (`X-Served-By`) on a brand new Ubuntu server.
- `1-install_load_balancer` : (To be added) Bash script to install and configure HAProxy to distribute traffic to web-01 and web-02 using round-robin.
